Once you actually have data within your database, or maybe you have a pre-populated database, it makes sense that the next step is knowing how to read from the database. In our case, this is how to read from a SQLite database in Python.

The code used here is just a simple example, as there are better and more efficient ways of reading from a database. More on that later. For now, assuming you have created the tutorial.db database from previous tutorials:

import sqlite3
import time
import datetime

conn = sqlite3.connect('tutorial.db')
c = conn.cursor()
sql = "SELECT * FROM stuffToPlot WHERE keyword =? AND source =?"

wordUsed = 'Python Sentiment'
sourceVariable = 'twitter'

def readData():
    for row in c.execute(sql, [(wordUsed), (sourceVariable)]):
        print str(row).replace(')','').replace('(','').replace('u\'','').replace("'","")
    

readData()
